# Vehicle Companion

An Android application for managing your vehicles and discovering nearby places of interest.

## Features

- **Garage:**
  - Add, edit, and delete your vehicles.
  - All vehicle data is persisted locally using the Room database.
- **Places:**
  - Discover and view details about nearby points of interest.
  - Fetches data from a remote API using Retrofit.

## Built With

- **UI:** Jetpack Compose
- **Architecture:** MVVM (Model-View-ViewModel)
- **Dependency Injection:** Hilt
- **Database:** Room
- **Networking:** Retrofit
- **Image Loading:** Coil
- **Language:** Kotlin
